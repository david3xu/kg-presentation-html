Pattern-Based Query Example

SPARQL Example (RDF):

PREFIX kg: <http://example.org/ontology/>

SELECT ?researcher ?publication ?field
WHERE {
  ?researcher a kg:Researcher .
  ?publication kg:hasAuthor ?researcher .
  ?publication kg:field ?field .
  ?field kg:name "Knowledge Graphs" .
}

Cypher Example (Property Graph):

MATCH (r:Researcher)-[:AUTHORED]->(p:Publication)-[:IN_FIELD]->(f:Field)
WHERE f.name = "Knowledge Graphs"
RETURN r.name, p.title

These queries demonstrate how graph databases allow for intuitive, relationship-focused querying, making complex questions straightforward to express.
